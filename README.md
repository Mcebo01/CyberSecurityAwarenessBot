## Overview
A console-based Cybersecurity Awareness Assistant built in C# .NET 8 that educates South African citizens about online safety.

## Features
- Voice greeting on startup
- Beautiful cybersecurity-themed ASCII art
- Personalised user interaction
- In-depth responses on phishing, passwords, 2FA, safe browsing
- Enhanced coloured console UI with typing effect
- Full input validation and graceful error handling
- Professional code structure (classes + methods)

## How to Run
1. Clone the repository
2. Open in Visual Studio 2022+
3. Build the solution (Ctrl+Shift+B)
4. Run (F5)

*Make sure greeting.wav is in the output folder.*
